รายละเอียดที่ 2.7

ใช้รูปแบบการบริหาร Dara Academy Team (DT Model) ในการบริหารและการจัดการ

เกณฑ์การพิจารณา ข้อ 6

กำหนดรูปแบบทีมบริหารโมเดล ประกอบด้วยทีมนำ ทีมพัฒนาคุณภาพ และทีมปฏิบัติ ในการบริหารและการจัดการ

เอกสาร/หลักฐาน/ร่องรอยการปฏิบัติ

  • แผนพัฒนาคุณภาพการศึกษา
  • แผนปฏิบัติการประจำปี
  • รายงานผลการดำเนินงานตามแผนปฏิบัติการประจำปี
  • งานวิจัย DT Model
  • แบบบันทึกการใช้ห้อง ICT
  • แบบบันทึกการใช้ห้องคอมพิวเตอร์ของวิชาการ
  • แบบบันทึกการใช้ห้อง PAS
  • แบบรายงานสื่อเทคโนโลยี
  • แบบบันทึกแหล่งเรียนรู้ภายในและภายนอกโรงเรียน (SAR)
  • ภาพความสำเร็จของผู้บริหาร

คำอธิบายการดำเนินงานตามเกณฑ์ (ข้อ 6)

 โรงเรียนดาราวิทยาลัยมีการบริหารจัดการอย่างเป็นระบบมีโครงสร้างการบริหารที่ชัดเจนโดยการบริหารจัดการที่ผู้เกี่ยวข้องทุกฝ่ายมีส่วนร่วมในการจัดการศึกษาภายใต้รูปแบบการบริหาร DT Model ประกอบไปด้วยทีมนำ ทีมพัฒนาคุณภาพ และทีมปฏิบัติ ในการที่จะประสานความร่วมมือในการปฏิบัติงาน โครงการ ตลอดจนกิจกรรมต่าง ๆ ควบคู่ไปกับการใช้วงจรการบริหารงานคุณภาพของเดมมิ่ง (PDCA) ปรากฏดังแผนภาพ

แผนภาพ DT Model / โครงสร้างทีมนำ-ทีมพัฒนาคุณภาพ-ทีมปฏิบัติ

รูปแบบการบริหาร DT Model (Dara Academy TEAM Model)
T : Technology หมายถึง สื่อ วัสดุ อุปกรณ์ เทคโนโลยีในการจัดการเรียนการสอน/บริหารที่หลากหลาย ทันสมัย เหมาะสมและเพียงพอ
E : Economic หมายถึง สมรรถนะด้านเศรษฐกิจ งบประมาณจากภาครัฐและเอกชน รวมถึงค่าเล่าเรียนของผู้ปกครอง
A : Attitude หมายถึง เจตคติที่ดีของนักเรียน ผู้ปกครอง ครูและบุคลากร รวมถึงบรรยากาศที่ดีในการทำงานร่วมกัน
M : Management หมายถึง ระบบบริหารจัดการด้วย PDCA และการบริหารโดยใช้โรงเรียนเป็นฐาน (SBM)

แผนภาพ นวัตกรรมการสอนเพื่อพัฒนาทักษะการคิด

 โรงเรียนดาราวิทยาลัย ได้มีการพัฒนารูปแบบการบริหาร DT Model ว่าด้วยการปฏิบัติงานเป็นทีมของครูโรงเรียนดาราวิทยาลัย จังหวัดเชียงใหม่ โดยมีกระบวนการของการปฏิบัติงานเป็นทีมของครูโรงเรียนดาราวิทยาลัยจังหวัดเชียงใหม่ แบ่งออกเป็น 5 ขั้นตอน ดังนี้
 1) ร่วมมือร่วมแรงคิด (iCooperation)
 2) ร่วมสร้างสรรค์ (iCreation)
 3) ร่วมมือทำ (iCollaboration)
 4) ร่วมเสริมแรงใจ (iCelebration)
 5) ร่วมเชื่อมสัมพันธ์แบ่งปันเรียนรู้ (iConnection)

แผนภาพ DT Model

 การพัฒนารูปแบบการบริหาร DT Model ว่าด้วยการปฏิบัติงานเป็นทีมของครูโรงเรียนดาราวิทยาลัย จังหวัดเชียงใหม่ โรงเรียนดาราวิทยาลัยได้ดําเนินการสังเคราะห์ กระบวนการของการปฏิบัติงานเป็นทีมของครูโรงเรียน ดาราวิทยาลัย จังหวัดเชียงใหม่ แบ่งออกเป็น 5 ขั้นตอน ได้แก่
1. ร่วมมือร่วมแรงคิด (iCooperation) สมาชิกมีส่วนร่วมในการกําหนดเป้าหมายในการทํางาน ผ่านการระดมความคิดเห็นเป็นการสร้างความ ตระหนักรับรู้ร่วมกันระหว่างสมาชิกในทีมให้เข้าใจถึงคุณค่าของการทํางานร่วมกัน โดยร่วมวางแผน พิจารณาตัดสินใจว่าจะทําอะไร อย่างไร กําหนดวัตถุประสงค์ นโยบาย โครงการ ผู้รับผิดชอบสําหรับการปฏิบัติการและคาดการณ์ผลที่จะเกิดจากการดําเนินการที่อาจเกิดขึ้นในอนาคต ขั้นตอนนี้สามารถใช้เทคโนโลยี Google Classroom, Google Sheet, Google Meet, Microsoft Team, Zoom, Padlet, Mentimeter, Application Line ฯลฯ ในการปฏิบัติงาน
2. ร่วมสร้างสรรค์ (iCreation) สมาชิกในทีมร่วมสร้างสรรค์แนวทางและวิธีปฏิบัติเพื่อให้บรรลุวัตถุประสงค์ โดยมีการวิเคราะห์ข้อมูล อย่างรอบด้าน เพื่อให้การขับเคลื่อนการปฏิบัติงานบรรลุผลตามเป้าหมาย บนพื้นฐานหลักปรัชญาของเศรษฐกิจพอเพียงขั้นตอนนี้สามารถใช้เทคโนโลยี Google Form, Padlet, Mentimeter, Google Sheet, Application Line ฯลฯ ในการปฏิบัติงาน
3. ร่วมมือทำ (iCollaboration) สมาชิกในทีมนําแนวทาง/วิธีการ มาสู่การปฏิบัติจริงตามที่ได้วางแผนไว้โดยสมาชิกทุกคนยึดถือปฏิบัติ ไปในทิศทางเดียวกัน เพื่อให้บรรลุตามเป้าหมายที่ได้ตั้งไว้ มีระบบการบริหารจัดการที่ดี มีการแบ่งบทบาท หน้าที่ความรับผิดชอบอย่างเหมาะสมและใช้ทรัพยากรอย่างคุ้มค่าและเกิดประโยชน์สูงสุดและประเมินผลการดําเนินงานเพื่อนํามาปรับปรุงพัฒนางานตลอดการดําเนินการขั้นตอนนี้สามารถใช้เทคโนโลยีเพื่อการจัดการศึกษาหรือการจัดการเรียนรู้ และเทคโนโลยีเพื่อการสร้างผลงานและนวัตกรรม ในการปฏิบัติงาน
4. ร่วมเสริมแรงใจ (iCelebration) การปฏิบัติงานร่วมกันเป็นทีมนั้น สิ่งที่ควรคํานึงถึงอยู่เสมอ คือ การให้กําลังใจในการทํางาน คําพูด ที่แสดงออกถึงความชื่นชมยินดี หรือคําพูดที่สร้างพลังใจให้แก่สมาชิก การให้รางวัลหรือสิ่งของเล็ก ๆ น้อย ๆตลอดจนการแสดงออกถึงความพึงพอใจต่อความพยายามในการทํางานของสมาชิก เพื่อมุ่งสู่เป้าหมาย โดยเฉพาะอย่างยิ่ง ผู้นําองค์กร/ผู้นําทีม ซึ่งถือได้ว่า เป็นผู้ที่มีบทบาทในการทําให้ทุกคนมีความรู้สึกว่าเป็นส่วนหนึ่งที่มีความสําคัญ ทั้งนี้เพื่อมุ่งให้การปฏิบัติงานของทีมประสบผลสําเร็จ ทําให้สมาชิกเกิดการเห็นคุณค่าของตนเอง(Self-esteem) ได้อย่างชัดเจน และสามารถพัฒนาศักยภาพตนเองได้อย่างไม่มีที่สิ้นสุดขั้นตอนนี้สามารถใช้เทคโนโลยี Application Line, web Page, VDO ฯลฯ ในการปฏิบัติงาน
5. ร่วมเชื่อมสัมพันธ์แบ่งปันเรียนรู้ (iConnection) สมาชิกในทีมสะท้อนผลการดําเนินงานทั้งภาพความสําเร็จและสิ่งที่ควรปรับปรุงพัฒนา เพื่อการแบ่งปัน แลกเปลี่ยนเรียนรู้ (Knowledge Sharing) เป็นการสร้างชุมชนแห่งการเรียนรู้ทางวิชาชีพ (Professional Learning Community : PLC) ให้เกิดขึ้นในองค์กร โดยทุกคนได้รับโอกาสในการถ่ายทอด แลกเปลี่ยนความรู้ประสบการณ์ และทักษะร่วมกัน ส่งผลให้องค์กรและบุคลากรมีกระบวนการทํางานที่มีประสิทธิภาพ และมีผลการปฏิบัติงานที่มีประสิทธิผล เพื่อเตรียมรับกับการเปลี่ยนแปลงที่อาจเกิดขึ้นในอนาคตได้ตลอดเวลา เป็นการพัฒนาที่นําไปสู่การเป็นโรงเรียนแห่งการเรียนรู้ (Learning Community) และขยายผลสู่องค์กรภายนอกสถานศึกษาขั้นตอนนี้สามารถใช้เทคโนโลยี Google Classroom, Microsoft Team, Zoom, Application Line, Web Page ฯลฯ ในการปฏิบัติงาน

สารบัญ

คลิกเพื่อไปยังหน้าข้อย่อยของหัวข้อ 2.3

  • ข้อ 1) มีการนำเทคโนโลยีมาใช้ในการสอนและปฏิบัติงาน
  • ข้อ 2) มีการบริหารงบประมาณ
  • ข้อ 3) สร้างเสริมเจตคติต่อโรงเรียนและการสร้างบรรยากาศการทำงานที่ดี
  • ข้อ 4) มีการบริหารจัดการด้านบุคลากร งบประมาณวัสดุอุปกรณ์ และการจัดการสภาพแวดล้อม
  • ข้อ 5) ผู้บริหารมีทักษะที่จำเป็นในศตวรรษที่ 21 เพื่อความสำเร็จขององค์กร (สื่อสาร / บริหารจัดการองค์กร / คิดวิเคราะห์และคิดสร้างสรรค์ / เทคโนโลยีและดิจิทัล / มนุษยสัมพันธ์)
  • ข้อ 6) กำหนดรูปแบบทีมบริหารโมเดล ประกอบด้วยทีมนำ ทีมพัฒนาคุณภาพ และทีมปฏิบัติ ในการบริหารและการจัดการ
